Структура узла Stack содержит два поля:

tags: массив символов, представляющий собой основу стека, куда будут помещаться элементы.
size: целочисленное значение, отражающее количество элементов в стеке.
Подпрограммы:

Push - процедура добавления элемента в стек. Если стек уже заполнен (размер равен MAXSIZE), операция завершается.
Pop - функция извлечения элемента из стека. Если стек пустой (размер равен 0), возвращается символ char(255 - пустой символ).
isEmptyStack - функция проверки стека на наличие элементов. Возвращает True, если размер стека равен 0, иначе False.
Особенности стека:
Стек работает по принципу "последний вошел - первый вышел" (Last-In-First-Out, LIFO). Это означает, что новые элементы добавляются на вершину стека и извлечение происходит также с вершины.

Код читает символы из файла 'input.txt', помещает их в стек и записывает в файл 'output.txt', при этом сохраняя порядок обработки (последний символ из файла становится первым в файле вывода).

В конце программы выводится сообщение 'копирование текста завершено'.
